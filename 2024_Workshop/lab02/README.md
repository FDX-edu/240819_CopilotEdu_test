# Lab 02: 기초 실습

## 이 랩의 목표 


 이 실습에서 다루는 주제는 다음과 같습니다. 
 -   메시지 및 질문 노드의 기본 사항을 알아봅니다.                                 
 -   entities 및 slot filling의 기본 사항을 발견하십시오.                       
 -   Microsoft Copilot Studio에서 변수를 사용하는 방법을 알아봅니다.              
 -   Power Fx 및 확장 노드 속성을 포함한 확장성 기능을 살펴봅니다.
